## 운영 규칙

- [필수알고리즘](/docs/essential%20algorithm.md)에 기제된 알고리즘을 주간 테마로 잡고 관련 문제 출제
  - 매주 월요일 공개

- 하루에 한문제 풀기
  - 필수 1개
  - 필수를 이전에 풀었을 경우를 대비하는 폴백 1개
  - 총 2개 중 1개 풀기
  - 매일 오전 중 공개

- 코드 리뷰 하기
  - 해당 [레포지토리](https://github.com/jadewisemann/morgorithm)에 맴버로 초대되어서 각자의 pr에 approve를 해야함
  - 전원 승인해야 pr되도록 설정
  - 물론 그냥 apprvoe만 해도 되지만, 성장을 위해서 남의 코드를 읽어버릇 하기

- 미팅은 주에 한번
  - 매주 월요일 18:30 에 모여서 한 문제 출제
  - 1시간 동안 풀고 30분 동안 풀이 공유 및 풀이

## 파일 업로드 규칙

### 파일 명

- 문제 풀이 파일은 **아래 형식을 반드시 지켜서 업로드**해 주세요.
- !! 형식에 어긋난 pr은 반려됨
  - 추후 봇 도입 예정

- 식별자는 알아서 정하기
  - 영어 소문자 및 숫자만 가능 (특수문자 사용 불가)
  - 숫자로 시작하지 않기
  - 한 단어 또는 약어로  

- 예
  ```
  백준문제번호_식별자.py
  ```

### 업로드 해야 할 폴더

- [problems](/problems/) 폴더 안에 맞는 주차
- 예를 들어 1주차면 [problems/week1](/problems/week1/)안에 [problems/week1/1000_jade.py](/problems/week1/1000_jade.py)와 같이 올리기
